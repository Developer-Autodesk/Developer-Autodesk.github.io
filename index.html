Libraries

Here are some libaries which make your work easly. Your are welcome to use them in your project. And we encourage you contributing to these libraries by forking and sending pull requests to make it better.

library-javascript-viewer-extensions
A collection of various JavaScript extensions for the viewer. library-javascript-view.and.data.api
This is a JavaScript library providing a high level wrapper for the Autodesk View & Data Rest API calls. library-dotnet-view.and.data.api
Autodesk View And Data Rest API Wrapper Library for .Net Applications and ASP.Net websites.

Other samples

Some additional samples in various languages, demonstrating the workflow of view and data API.

  • workflow-node.js-view.and.data.api
    Basic Node.js server and client for the Autodesk View and Data API
  • workflow-macos-swift-view.and.data.api
    MacOS sample written in Swift
  • workflow-wpf-view.and.data.api
    A WPF sample that illustrates what is doable with the Autodesk View and Data API
  • workflow-dotnet-winform-view.and.data.api
    This is a sample for WinForm desktop application demoing the partial workflow of using Autodesk View and Data API
  • workflow-android-view.and.data.api
    This is a sample for Andriod devices demoing the partial workflow of using Autodesk View and Data API
  • workflow-ios-view.and.data.api
    This is a sample for iOS devices demoing the partial workflow of using Autodesk View and Data API
  • workflow-curl-view.and.data.api
    A sampple demoing the workflow of using View and Data API in command line tool.

Here are some other samples, demonstrating the integration with other systems, such as your own ERP/CRM system, exteneral Database, etc.

  • integration-sap-view.and.data.api
    An ASP.Net MVC website sample that integrates the Autodesk View and Data API with an SAP database using the   SAP Gateway for Microsoft (GWM)   technology.
  • integration-bing-maps-view.and.data.api
    This is a sample demoing the integration with Microsoft Bing Maps and Autodesk View and Data API.
  • integration-sharepoint-view.and.data.api
    This repo contains files that show how you can upload files and view them inside SharePoint using the Autodesk viewer.
