View and Data API

Embedding vibrant 3D graphics in a Web Browser, Support for almost 100 formats, Fast streaming large 2D and 3D models

What exactly does the View and Data API do?

The View and Data API enables web developers to very easily display 3D (and 2D) models on a WebGL-enabled browser. If you've got a model you've created using some kind of design software, then you can probably display it on your website using this API.

Play and learn the APIs

Pretty exciting, right? Eager to learn some APIs and get it running on your own website? Here are some live samples you can play with to understand the workflow and some client APIs.

The API comes in two parts -

  • A (server-side) REST API that allows you to translate any of almost 100 design file formats into a common (JSON) file format. The data extracted from the original file is the geometry of every element in the model and the metadata that was attached to each of those model elements.
  • A (client-side) JavaScript API that simplifies displaying and interacting with a model that has been processed by the translator. The UI for the viewer defined by the API can be heavily customized, viewer actions can be automated (zoom, pan rotate, explode, etc), and callbacks are available for user-driven events (e.g. a mouse click on a specific model element, a change in camera position, etc).

You are encouraged to run following live applications first to understand the concept of workflow with server side REST API, practice some Client side JavaScript APIs, and then setup your first 3D model viewer web page with the utility live sample.
