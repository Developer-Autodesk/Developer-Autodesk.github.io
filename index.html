View and Data API

Welcome to Autodesk View and Data API developer center

The View and Data API is a web service that lets you display your 3D content on your website. Here's a quick start guide to help you started.

Play with it live

  • Manufacturing Sample
  • AEC Sample

Getting Started

To access the view and Data API, you need to register an App on http://developer.autodesk.com to get your consumer key and secret key. Register now!

Libraries

Here are some libaries which make your work easly. Your are welcomce to use them in your project. And we encourage you contributing to these libraries by forking and sending pull requests to make it better.

  • library-javascript-viewer-extensions
    A collection of various JavaScript extensions for the viewer
  • library-javascript-view.and.data.api
    This is a JavaScript library providing a high level wrapper for the Autodesk View & Data Rest API calls
  • library-dotnet-view.and.data.api
    Autodesk View And Data Rest API Wrapper Library for .Net Applications and ASP.Net websites
